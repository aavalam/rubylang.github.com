Some Top Ruby Projects

  • Rails Rails is a web application framework that includes everything needed to create database-backed web applications according to the Model-View-Controller (MVC) pattern.
  • Homebrew Homebrew is the easiest and most flexible way to install the UNIX tools Apple didn't include with OS X.
  • Jekyll Jekyll is a blog-aware, static site generator.
  • Diaspora A privacy aware, personally controlled, do-it-all, open source social network.
  • Travis CI Travis is an open, distributed build system for the Ruby community.
